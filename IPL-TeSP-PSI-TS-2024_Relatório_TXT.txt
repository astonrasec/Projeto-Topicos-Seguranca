1 INTRODUÇÃO 
Este projeto tem como objetivo o desenvolvimento de um sistema de chat seguro em C#, implementando mecanismos de criptografia (simétrica e assimétrica), autenticação e validação de dados através de assinaturas digitais. O sistema será composto por um módulo cliente com interface gráfica (Windows Forms) e um módulo servidor console, permitindo comunicação segura entre múltiplos clientes. 
A segurança das comunicações será garantida através do uso de algoritmos criptográficos, sockets TCP/IP e protocolos de autenticação robustos.
2 ESPECIFICAÇÃO DO SISTEMA
O problema a resolver é a necessidade de um sistema de troca de mensagens seguro onde a confidencialidade, integridade e autenticação dos dados são críticas. 
O sistema deve permitir que múltiplos clientes se conectem a um servidor central, autentiquem-se com credenciais, e troquem mensagens cifradas de forma segura. 
O servidor deve validar todas as assinaturas digitais das mensagens, armazenar as chaves públicas dos clientes e manter um registo seguro de todas as operações.
